A.R.C.E.U.S

{ Automated Responsive Userfriendly Structure for ECommerce }

ARCEUS is a framework for E-Commerce, developed in PHP which focuses its goals on automating the process of ECommerce developement.It enables the user to create a ECommerce site with no knowledge of coding.
